# MVC Framework Server Router Class
MVC Framework Server Router Class configures Fetch API for transactions between a client and server. 

## Settings
```
{
  scheme: String,
  domain: String,
  port: Number
  routes: Object {
    [$resourceName]: {
      path: $resourcePath,
      methods: {
        [$methodName]: $methodOptions
      }
    }
  }
}
```
